# Gaucho Guardian
Gaucho Guardian is a tool for UCSB Students to help them reduce the strain of selecting classes. This is a SB Hacks XII project.

TO RUN:
- Install Poetry for Python
- Run 'poetry install'
- Run 'export UCSB_API_KEY="INSERT_KEY"' (You can request an api key from developer.ucsb.edu)
- Run 'poetry run flask run'
